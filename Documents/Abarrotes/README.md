# Listas de Compra - Abarrotes

Sistema administrativo diseñado para la gestión eficiente de pedidos diarios en la empresa Ometeotl.

## 🚀 Características
- **Diseño Responsivo:** Adaptado para móviles y computadoras.
- **Modo Offline:** Funciona sin conexión a internet gracias a librerías locales.
- **Gestión de Fases:** Flujo de trabajo dividido para un control preciso de pedidos.
- **Autónomo:** Configurado para arrancar y apagarse automáticamente en entornos locales.

## 🛠 Tecnologías utilizadas
- **Backend:** Java 21 + Spring Boot.
- **Frontend:** HTML5, Bootstrap 5.3 (Local).
- **Base de Datos:** SQLite.
- **Herramientas:** Git, Maven, NetBeans.

## 💻 Instrucciones de ejecución
1. Asegúrese de tener instalado el JDK 21.
2. Ejecute el archivo `Iniciar_Ometeotl.bat` en su terminal.
3. Acceda desde su navegador a `http://localhost:8085`.

## 📖 Guía de Usuario
- **Selección:** Elija fecha y proveedor.
- **Agregar:** Ingrese productos a la lista.
- **Validación:** Marque los estados de Aceptado y Recibido.
