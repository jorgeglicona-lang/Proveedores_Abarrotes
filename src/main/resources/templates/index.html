¿Cómo usar esta app?
  • 1. Selección: Elija la fecha de compra y proveedor para cargar la lista correspondiente.
  • 2. Agregar: Escriba el producto y presione "AGREGAR" para incluirlo en la lista.
  • 3. Verificación: Marque los cuadros según el avance "Aceptado" o "Recibido" (opcional).
  • 4. Finalizar: Use los botones inferiores para avanzar a la siguiente fase de revisión. Los botones apareceran despúes de agregar el primer producto a la lista.
